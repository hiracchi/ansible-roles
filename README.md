# ansible-roles

## usage
Link this directory to `~/.ansible/roles` before using:

```
$ ln -s `pwd` ~/.ansible/roles
```

or check Environment Variable; ANSIBLE_ROLES_PATH


## develop a new role

```
ansible-galaxy init  new_role
```
